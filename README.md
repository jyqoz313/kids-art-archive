## 这是一款什么样的App

闪闪童画是 besttime.us 出品的一款儿童画作数字化收纳工具。说人话就是：用手机扫一扫孩子的纸质画，App会自动对焦、自动裁边，把它变成一张高清电子档；然后你可以给它选个画框，标上标题、材质和日期，存进按主题或年份分好的画册里。

更妙的是，它还能生成一个线上画展链接，发给爷爷奶奶外公外婆，让他们在手机上就能逛孙辈的"个人美术馆"。

开发者 XiaoFeng 是成都的一位全栈工程师，做这个App的初衷就是为了保存自己儿子的画。这种"爸爸为儿子写软件"的故事，听着就让人觉得靠谱几分。
